# giwow projects

## Usage

```
Usage:
  giwow projects [flags]
  giwow projects [command]

Available Commands:
  add         Manage projects in the current workspace
  clone       Manage projects in the current workspace
  discover    Try to discover projects in the current workspace
  open        Open a project with the defined editor
  prune       Remove all projects from the current workspace

Flags:
  -h, --help           help for projects
      --netrc string   The path to the netrc file, default is $HOME/.netrc
```
