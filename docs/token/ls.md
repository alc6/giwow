# giwow token ls

## Usage

```
This will basically list your tokens in your $HOME/.netrc file.

Usage:
  giwow token ls [flags]

Flags:
  -h, --help       help for ls
  -m, --magnify    Print tokens in a magnified array format
      --showPass   Display your tokens passwords
```
